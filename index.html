Conoce un poco más sobre este proyecto

¿Qué es?

Un formulario web que permite enviar mensajes de contacto y guardarlos en una base de datos MySQL.

Tecnologías

Está desarrollado con HTML, CSS, JavaScript y PHP. Los datos se almacenan usando XAMPP o InfinityFree.

Instrucciones

Completa el formulario y haz clic en “Enviar mensaje”. Recibirás una confirmación instantánea.
